# Claude Code Masterclass

This is a learning environment. The user may be new to programming.

## Guidelines

- Explain what you're doing and why, in simple terms
- Write clean, readable code with comments where helpful
- Start with simple implementations, then add complexity when asked
- If the user's request is vague, ask clarifying questions before building
- Prefer simple tech stacks: vanilla HTML/CSS/JS, Python scripts, or Node.js
- When creating web apps, include a way to run them (e.g., a simple HTTP server)
